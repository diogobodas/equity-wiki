## Workflow

1. Drop raw files (PDF, articles, data) into `undigested/`
2. Run ingest — LLM reads source, creates/updates wiki pages
3. LLM generates exhaustive summary in `digested/{name}_summary.md`
4. Original file is deleted from `undigested/`
5. Entry below is updated with status = digested

If the summary is insufficient for a future task, provide the original file again in `undigested/` for re-extraction.
